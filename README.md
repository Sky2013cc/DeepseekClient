## 🛠️ Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.20.1
2. Download the latest DeepseekClient `.jar` from [Releases](https://github.com/Sky2013cc/DeepseekClient/releases)
3. Place the `.jar` in your `mods` folder
4. Launch Minecraft with Fabric profile

## 🔧 Building

```bash
git clone https://github.com/Sky2013cc/DeepseekClient.git
cd DeepseekClient
./gradlew build
# Output: build/libs/DeepseekClient-*.jar
```

## 📜 Commands

All commands use the `.` prefix. Type `.help` in-game for full list.

```
.toggle <module>    - Toggle a module on/off
.bind <module> <key> - Bind a key to a module
.friend add/remove/list <name> - Friend management
.modules           - List all modules
.config save/load  - Save/load configuration
.help              - Show help
```
